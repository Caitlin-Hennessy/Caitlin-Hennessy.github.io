Lipstick Analyzer

Using Bayesian analysis to predict average ratings of MAC lipsticks

As a makeup enthusiast, I love going to the MAC store and finding a great new lipstick. But as a scientist, I never go without researching the product reviews first. One of the biggest challenges when reading reviews on websites such as MakeupAlley is that it's easy to identify the most popular shades and the top rated shades. But I want to know what the best shades are, where "best" takes into account both the number of reviews and the average rating.

Global Centroid Moment Tensor data visualizer

Interactive map for visualization of earthquake data from the GCMT Catalog

Collaborated with a group of three software developers and one geology expert to convert a static-HTML page into a web app based on Node.js, Express, and MongoDB. We converted the earthquake data to GeoJSON format for ease of querying. Rearchitecting the project this way reduced the average page load time by 500%. For consistency between production and development environments, we created Docker containers for the frontend code and database, using docker-compose to link them together. See the app live here.
